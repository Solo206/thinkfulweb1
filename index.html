How to style with CSS

There are 3 ways to style:

  • external
  • internal
  • inline
external styling
External styling is ideal when the style is applied to many pages. With an external style sheet, one file controls the entire look of the website. <link rel="stylesheet" href="stylesheet.css">
internal styling
Internal styling is used when a single document has a unique style. Style is defined in the head section of an HTML page <style> div{color:white;}</style>
inline stying
Inline styling loses many of the advantages of external style sheet, by mixing content with presentation. To use inline styles, add the style attribute to the relavant tag. Style attribute can contain any CSS property. <p style="color:white;margin-left:20px">This is a style.</p>

Styling occurs in the following precendence order from lowest to highest:

  1. Browser default
  2. External styling
  3. Internal styling
  4. Inline styling

Block-level elements

  • If no width is set, it will expand naturally to fill its parent container
  • Can have margins and or padding
  • If no height is set will expand naturally to fit its child elements
  • By default will be placed below previous elements in the markup assuming no floats or positioning on surrounding elements

Inline elements

  • Flows along with text content
  • Will not clear previous content to drop to the next line like block elements
  • Is subject to white space settings in CSS

Block vs Inline

Block elements begin on a new line stacking one on top of the other and occupy any available width. Block-level elements may be nested inside one another and may wrap inline-level elements. We'll most commonly see block level elements used for larger pieces of content such as paragraphs.

Iinline do not begin on a new line. They fall into normal flow of the document, lining up one after the other, and only maintain the width of their of their content. Inline level elements may be nested inside one another; however they cannot wrap block-level elements. We'll usually see inline level elements with smaller pieces of content.

Headings

Headings are block-level elements. Headings are also used to establish hierarchy, they are key identifiers used by users. Search engines also use them to index and determine content of page. Headings hsould be used in order that relavant to content of page. Subsequent headings within h1 sphere should use h2, h3, h4, h5, and h6. Headings have semantic value and should not be used to increase size of fonts.

Paragraphs

Headings are also followed by supporting paragraphs. Paragraphs are defined using the p block element.

Bold Text with Strong

Bold text places strong importance to it. Strong inline level element is used to give strong importance to text. <b> is used to stylistically offset text, for things that are not as important, but more for keywords. Gauge the usage based on the significance of the contextual meaning of words. If its actually important, defer to strong tag rather than b tag.
